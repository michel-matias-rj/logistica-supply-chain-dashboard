## 📖 Sobre o Projeto
Este projeto apresenta uma solução de Business Intelligence focada na otimização da cadeia de suprimentos (Supply Chain), permitindo o monitoramento de KPIs críticos para a operação logística.

## 🚀 Funcionalidades e KPIs
- **Gestão de Nível de Serviço (SLA):** Controle de % de atrasos.
- **Eficiência de Custos:** Consumo de combustível e média por viagem.
- **Visibilidade de Rede:** Faturamento detalhado por galpão/unidade.
- **Produtividade:** Ranking de performance de motoristas.

## 🛠️ Tecnologias Utilizadas
- **Power BI:** Extração, Modelagem e Visualização.
- **DAX:** Criação de medidas de inteligência de negócio.
- **Power Query:** Tratamento e limpeza de dados (ETL).
